مستندات API

تمام endpoint های موجود در api.php اینجا شرح داده شده‌اند. همه درخواست‌ها نیاز به Bearer Token دارند و پاسخ‌ها به فرمت JSON برگردانده می‌شوند.

معرفی
Base URL
https://api.gozargah.network/api.php
Content-Type
application/json; charset=utf-8
متدهای مجاز
GETPOSTPATCHDELETE
فرمت پاسخ
JSON

هر درخواست باید فیلد action داشته باشد. مقادیر مجاز: create_order · update_order · delete_order · get_user · deduct_wallet

احراز هویت
🔑
Bearer Token — همه درخواست‌ها باید هدر Authorization با مقدار Bearer <token> داشته باشند.
Authorization: Bearer YOUR_SECRET_TOKEN
⚠️
قبل از deploy حتماً توکن را به یک مقدار امن تغییر دهید.

POST https://api.gozargah.network/api.php کسر از کیف‌پول

مبلغی را از کیف‌پول کاربر کسر می‌کند. این عملیات با تراکنش پایگاه داده و قفل سطر (FOR UPDATE) انجام می‌شود تا از race condition جلوگیری شود.

⚠️
توجه: این endpoint موجودی را بررسی نمی‌کند — کیف‌پول می‌تواند منفی شود. اعتبارسنجی موجودی باید در لایه اپلیکیشن انجام شود.
